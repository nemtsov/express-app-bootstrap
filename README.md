Express App Bootstrap
=====================

The script bootstraps an Express-based node.js RESTful app.

Installation
------------
    npm -g install express-app-bootstrap


Usage
-----
    
    express-app-bootstrap new_app_name
    cd new_app_name
    npm install
    node app
